# GIMidi Player

Recomiendo utilizar [uv](https://github.com/astral-sh/uv) para manejar y ejecutar el programa. Una vez instalado, ejecutar:

```console
$ uv sync
$ uv run .\main.py path/to/your/midi/file.mid
```
